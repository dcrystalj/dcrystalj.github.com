How to become a blockchain developer — the first thing

tl;dr Blockchain A-Z™: Learn How To Build Your First Blockchain

I passed some online courses and read a lot of resources to become blockchain developer, even 100x more expensive ones. The first thing you need is to get basic idea how blockchain works, what are its use cases and what are its limitations in order to understand documentation of blockchain you are interested in. Only after that you can develop smart contracts you need and Blockchain A-Z course will give you all of that: basic background and some idea about smart contracts on Etherium blockchain. Keep in mind there are 2 types of blockchain developers:

  • Core blockchain developer
  • Blockchain software developer

Most of blockchain developers are the latter and you are probably looking for that as well. The former requires full understanding of how specific blockchain works while other don’t. It’s like comparing knowing how java virtual machine works and how to write an app in java.
